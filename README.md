# 🚗 Sistema de Gestión Vehicular – Proyecto Java Swing

Este proyecto es un **sistema de gestión vehicular** desarrollado en **Java Swing** como práctica de programación orientada a objetos (POO).

Actualmente permite gestionar **carros y motos** mediante un CRUD (Crear, Leer, Actualizar, Eliminar) y guarda los datos en archivos de texto (`.txt`) generados en cada ejecución.

---

## 📌 Funcionalidades
- Añadir vehículos (carros y motos)  
- Modificar información de vehículos  
- Eliminar vehículos  
- Listar todos los vehículos registrados  

> ⚠️ Nota: Este proyecto **no utiliza bases de datos**. Todos los datos se almacenan en archivos `.txt` generados durante la ejecución del programa.

---

## 🛠 Tecnologías
- Java (nativo)  
- Swing para la interfaz gráfica  
- Conceptos básicos de programación orientada a objetos (POO)  

> No requiere instalación de tecnologías adicionales, solo tener Java instalado.

---

## 📂 Estructura del proyecto
- `src/` – Código fuente en Java  
- `Vehiculos.txt` – Archivo generado durante la ejecución  
- `build/` – Carpeta de compilación (ignorada por Git)  
- `.gitignore` – Archivos y carpetas a ignorar
